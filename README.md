# Pulmuone TSR Dashboard

Static dashboard for annual TSR comparison of Pulmuone and selected Korean food peers.

The site includes:
- `index.html`: interactive chart and tables
- `raw-data.csv`: annual raw data table
- `tsr-data.json`: calculation data used by the site

Calculation basis: annual TSR is calculated as `(ending price - beginning price + dividend per share) / beginning price`. Adjusted close and dividend reinvestment adjustments are not used.
